🚀 GitHub Setup A-Z (Hinglish Guide)

1. Zaroori Cheezein (Requirements)

  • GitHub Account
  • GitHub CLI (gh)
  • Git
  • VS Code (optional)

2. SSH Key Setup

Terminal mein yeh likho:

ssh-keygen -t ed25519 -C "youremail@example.com"

Phir key dekhne ke liye:

cat ~/.ssh/id_ed25519.pub

Is key ko yaha paste karo: Add SSH Key on GitHub

3. GitHub CLI Login

gh auth login

Select: GitHub.com → SSH → Browser se login karo

4. Aliases (Shortcut Commands)


alias gitstart='git init && git add . && git commit -m "🚀 First commit"'

alias gitall='git add . && git commit -m "🧠 Auto Save" && git push origin main'

alias mkrepo='
  read -p "📁 Repo Name: " reponame;
  read -p "🔒 Private or Public? (private/public): " visibility;
  gh repo create "$reponame" --$visibility --source=. --remote=origin --push
'

alias mkpages='
  gh repo edit --add-topic github-pages;
  gh api -X POST repos/:owner/:repo/pages --field "source[branch]=main" --field "source[path]=/";
  echo "✅ GitHub Pages enabled!";
  echo "🔗 https://$(gh repo view --json nameWithOwner -q ".nameWithOwner" | sed "s/\\\"//g").github.io"
'

5. Daily Workflow (A-Z)


cd ~/
mkdir mera-project && cd mera-project
code .

gitstart

mkrepo
# 📁 Enter repo name → jaise: my-website
# 🔒 Enter visibility → public ya private

git branch -m master main
git push -u origin main

mkpages

Yeh sab karne ke baad tera site live ho jayega!
